# BugBounty Setup

Command to install in Ubuntu/Kali Box


```
Step 1 Install dot files
wget -O- https://raw.githubusercontent.com/Gfuen/HackTheBoxSetup/main/dotfilesetup.sh
dos2unix dotfilesetup.sh
sudo bash dotfilesetup.sh
```

```
Step 2 Install Zsh and configurations
wget -O- wget https://raw.githubusercontent.com/Gfuen/HackTheBoxSetup/main/zshinstall.sh
dos2unix zshinstall.sh
sudo bash zshinstall.sh
```

```
Step 3 Install tools needed
wget -O- https://raw.githubusercontent.com/Gfuen/HackTheBoxSetup/main/Install.sh
dos2unix Install.sh
sudo bash Install.sh
```
